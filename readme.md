# PromptUI

Turn ideas into beautiful web interfaces using AI.

PromptUI is a **frontend-only generative AI project** that converts natural language prompts into **React + Tailwind CSS UI components** with a **live preview and code viewer**.

It allows developers and designers to quickly prototype modern UI layouts by simply describing them.

---

## Live Demo

You can see the project live here:

https://promptui.vercel.app

---

## Features

* Generate UI using natural language prompts
* Live UI preview
* React JSX code generation
* Syntax highlighted code viewer
* Prompt templates for faster generation
* AI example gallery
* Interactive preview environment
* Modern glassmorphic UI design
* Navbar navigation with hash routing
* Custom example screenshots / videos
* Production ready for Vercel deployment

## Installation

Clone the repository

```
git clone https://github.com/TusharMokal01/prompt-ui.git
```

Navigate into the project

```
cd ai-ui-builder
```

Install dependencies

```
bun install
```

Run development server

```
bun dev
```

---

## Environment Setup

Create a `.env` file in the root directory.

```
VITE_OPENROUTER_API_KEY=your_api_key_here
```

You can obtain a free API key from:

https://openrouter.ai

---

## Usage

1. Open the generator page
2. Enter a UI prompt
3. Click **Generate UI**
4. AI returns a **React JSX component**
5. View the generated UI in:

   * Live preview
   * Code viewer

You can copy the generated code and use it directly in your projects.

## Deployment

This project can be deployed easily using **Vercel**.

Steps:

1. Push repository to GitHub
2. Import the project into Vercel
3. Add environment variable:

```
VITE_OPENROUTER_API_KEY
```

4. Deploy
